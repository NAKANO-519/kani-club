# かにクラブ Webサイト

## 概要
職業訓練校のチーム制作課題として作成したWebサイトです。
福井県にある「越前がにミュージアム」の公式サイトリニューアルをコンセプトに制作しました。
チームで役割分担を行い、企画からデザイン・コーディングまで協力して制作しました。
ユーザーが情報を探しやすいレイアウト構成を意識しました。

## 使用技術
- HTML
- CSS
- JavaScript
- Adobe Photoshop
- Adobe Illustrator 

## 制作目的
Web制作の基礎（レイアウト構築・デザイン・動きの実装）を実践するため。
